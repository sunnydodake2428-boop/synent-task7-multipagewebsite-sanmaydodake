A complete multi-page website built as part of the **Synent Technologies Web Development Internship Program**

## 📋 About
**EdgeWire** is a freelance developer services site — built by me, Sanmay Dodake — showcasing full-stack development, AI integration, and UI engineering services. It's structured as a real multi-page site with consistent navigation, shared design system, and a functional contact form.

## 📄 Pages
- **Home** (`index.html`) — Hero section, quick stats, services preview, and a featured projects showcase linking to my live deployed work
- **About** (`about.html`) — Background, journey timeline, and technical skills
- **Services** (`services.html`) — Detailed service offerings and a 4-step working process
- **Contact** (`contact.html`) — Contact info cards and a working `mailto:`-based contact form

## ✨ Features
- Fully responsive across desktop, tablet, and mobile
- Consistent nav (with working mobile hamburger menu) and footer across all pages
- Active-page indicator in navigation
- Premium glassmorphism UI — frosted glass cards, gradient accents, ambient glow effects
- On-load staggered entrance animations, pure CSS (no JavaScript)
- Functional contact form (opens visitor's email client, pre-filled)

## 🛠️ Built With
- HTML5
- CSS3 (custom properties, Grid, Flexbox, backdrop-filter, keyframe animations)
- Google Fonts (Space Grotesk, Inter, JetBrains Mono)

## 🎨 Design System
- **Theme:** Dark, glassmorphic, blue/cyan gradient accents
- **Typography:** Space Grotesk for headings, Inter for body text
- **Components:** Reusable glass cards, gradient buttons, pill tags — consistent across every page
